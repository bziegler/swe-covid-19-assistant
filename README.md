<!-- ABOUT THE PROJECT -->
## About The Project

This project is a proof of concept for a semantic search engine of Sympathy for Data documentation using a pre-trained BERT transformer model. 

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install all the required packages for SENS by running the following:

```
pip install -r requirements.txt
```

### Installation
 
1. Clone the repo
```
git clone https:://github.com/github_username/repo.git
```
2. Install SENS
```
python ./setup.py
```



<!-- USAGE EXAMPLES -->
## Usage

1. Extract all documentation from Sympathy for Data
```
python ./get_docs.py
```
2. Run Flask app
```
python ./app.py
```
